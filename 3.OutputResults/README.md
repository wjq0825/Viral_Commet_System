### 📌 Post 001 - 职场竞争话题（严肃且敏感）

```

标题: 为什么现在的年轻人都在卷？
内容: 工作996，年薪30万起。还在抱怨的为什么不去试试？看到有人月薪10万，我就在想为什么自己不行。
形式: 纯文本
互动: 892评论
```

**A1 帖子理解**：
- 🎯 核心话题：职场竞争与焦虑
- 🔑 关键钩点：996工作制、年薪数字、卷文化、代际竞争
- 😤 情感语气：竞争焦虑
- 🚨 风险等级：5/10（中等风险 - 涉及职场、收入、阶层）

```

  示例评论: "年薪30万起，但扣税、房租、生活成本后，你根本攒不了钱"
  
套路2: 降维打击 - 用简单逻辑击穿复杂问题
  示例评论: "问题不是你不够努力，而是机制设计让卷成了唯一出路"
  
套路3: 代言群众 - 说出大多数人的心声
```
