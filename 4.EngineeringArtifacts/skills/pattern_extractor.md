# Skill: Pattern Extractor

## Description
从高赞/高互动参考评论中提取可复用的表达套路。

## Trigger
当获取到检索到的参考评论列表时触发。

## Inputs
- `reference_comments`: 包含评论文本、点赞数、平台来源的列表。

## Steps
1. **去噪**：剔除无意义的表情符号或纯语气助词。
2. **结构拆解**：分析评论的修辞手法（反转、比喻、讽刺、提问）。
3. **模式抽象**：将具体内容替换为变量，提取通用句式。
    - *示例*：从"我以为他在大气层，结果他在地下一层"提取出 "期待与现实的降维打击对比"。
4. **有效性评估**：说明该模式为什么在特定语境下有效。

## Output Template
```json
[
  {
    "pattern_name": "...",
    "formula": "...",
    "example": "...",
    "why_it_works": "..."
  }
]
```
